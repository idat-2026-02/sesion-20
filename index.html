TEMA 1

Estructuración Bidimensional con CSS Grid

Maquetación 2D

A diferencia de los modelos lineales de bloque o Flexbox, **CSS Grid Layout** opera de manera **bidimensional**, permitiéndote controlar simultáneamente las filas y columnas de un contenedor mediante código.

Contenedor Grid: Se activa declarando display: grid;. Define una matriz espacial para sus hijos directos.
Unidad fr (Fraction): Una unidad de medida flexible que representa una fracción del espacio libre disponible.
gap: Propiedad para definir el espaciado (canaleta) físico entre las filas y columnas de la rejilla.
Sintaxis de Rejilla
.contenedor-grid {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr; /* 3 columnas */
  grid-template-rows: auto;
  gap: 16px; /* Espaciado de celdas */
}
Dato clave: CSS Grid permite reordenar el layout visual sin tocar el código fuente del documento HTML. Maquetación Fluida

TEMA 2

Media Queries con Filosofía Mobile-First

Maquetación Adaptativa

En la metodología **Mobile-First**, la maquetación se define inicialmente para dispositivos móviles (pantalla estrecha) y se escala progresivamente hacia pantallas de escritorio usando condiciones min-width.

Código Base (Móvil): Todo el CSS fuera de las Media Queries se escribe asumiendo una visualización en una sola columna vertical.
Punto de ruptura min-width: 1020px: Al cruzar este límite, el navegador inyecta propiedades adicionales para reorganizar y expandir las columnas horizontalmente en pantallas anchas.
Ventaja técnica: Reduce drásticamente la redundancia de código y optimiza el rendimiento del renderizado en dispositivos portátiles.
Sintaxis Mobile-First

// 1. Estilos por defecto (Para móviles y tablets pequeñas)

.grupo-elementos {
  display: flex;
  flex-direction: column; /* Flujo apilado verticalmente */
}

// 2. Ampliación progresiva para pantallas grandes

@media (min-width: 1020px) {
  .grupo-elementos {
    flex-direction: row; /* Se amplía a formato horizontal */
  }
}
Dato técnico: Al utilizar Mobile-First, la condición "min-width: 1020px" solo se gatillará si el viewport es igual o superior a dicho ancho. Mobile Ready

TEMA 3

El Framework CSS Bootstrap

Frameworks CSS

**Bootstrap** es el framework de utilidades CSS más popular del mundo. Agiliza la construcción de interfaces responsivas gracias a su **sistema de grilla estructurado en 12 columnas**.

.container: Clase envolvente que centra el contenido y define el ancho máximo responsivo de la página.
.row (Renglón): Crea una fila flexible horizontal. Actúa como el contenedor de las columnas Bootstrap.
.col-* (Columnas de 12 Unidades): Define el espacio de ancho de un elemento sobre una base de 12 unidades (ej: col-md-4 ocupa 4 de 12 unidades, es decir, el 33.3%).
Sintaxis de Bootstrap Grid
<div class="container">
  <div class="row">
    <div class="col-md-6">Izquierda (50%)</div>
    <div class="col-md-6">Derecha (50%)</div>
  </div>
</div>
Dato clave: El sistema de grilla de Bootstrap utiliza Flexbox en su núcleo de renderizado para gestionar los alineamientos de las columnas. Estándares de Frontend

TALLER COMPARATIVO

¿Qué herramienta elegir?

Toma de Decisiones

📊 CSS Grid Puro

Excelente para maquetas completamente asimétricas, galerías de mosaicos complejas, diseño de tableros y control absoluto de los ejes X e Y sin elementos de envoltura innecesarios.

Sugerencia: Layouts únicos y personalizados.

📱 Media Queries

Recomendado cuando necesitas micro-ajustes específicos de diseño a nivel de tipografía, visualización selectiva de elementos, o adaptaciones hiper-específicas de componentes nativos.

Sugerencia: Microajustes responsivos específicos.

⚡ Bootstrap Grid

La mejor alternativa para proyectos comerciales estándar de alta velocidad. El sistema de grilla responsiva ya está pre-testeado, lo que te garantiza compatibilidad completa de navegadores de forma inmediata.

Sugerencia: Prototipado y lanzamientos rápidos.
Dato de la industria: El desarrollo frontend moderno de calidad frecuentemente hibrida CSS Grid interno con envolturas responsivas. Buenas Prácticas
